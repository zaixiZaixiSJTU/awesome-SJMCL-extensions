[SJMCL](https://github.com/UNIkeEN/SJMCL) 是一款面向 SJTU Minecraft Club 社区的 Minecraft 启动器。本仓库收录社区维护的 SJMCL 扩展，涵盖功能集成、界面增强、生产力工具等各类插件。欢迎通过 PR 添加你的项目，提交前请先阅读 [CONTRIBUTING.md](CONTRIBUTING.md)。
